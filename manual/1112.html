Operation Guide

Power On

There is a silicone power button on the top of the eEGG. Press this button firmly to power on the device.

‚ö†Ô∏è Caution: After receiving the eEGG, please fully charge the device before use.

Shake-and-Do Operation

The eEGG has a built-in G-sensor and can be used to achieve different functions by swinging it. Gently shake the eEGG several times, then stop and wait for about 1 second. The device will recognize the number of shakes and execute the corresponding function:

Operation Function Description
Shake 1 time Next/+ Switch to the next menu item; Play the next media file; Increase the volume.
Shake 2 times Run/Prev./- Execute the selected menu or function; Play the previous file; Decrease the volume.
Shake 3 times Open Menu Display the menu.
Shake 4 times Mode/Volume Shaking 4 times opens a pop-up menu for adjusting volume/playback mode;
Shake 5 times Power Off After powering off, press the button on the eEGG to power it on.

Charging

1

This device can be charged using various chargers with a Type-C interface. Under normal circumstances, the internal battery can be fully charged within 90 minutes.

2

If the device has been left unused for an extended period, causing over-discharge of the internal battery, please plug in the charger for 10 minutes before trying to turn the device on.

3

If the device is not used for a long time, it is recommended to fully charge it at least once every two months.

WiFi Configuration

1

Download ESP-Touch app:

ESP-Touch for Android ESP-Touch for iOS

2

Start "WiFi config" on eEGG:

  • For eEGG5, flick the device 3 times to display the menu, then flick once to switch to the next menu item, until you switch to the WiFi menu, then flick twice.
  • For eEGG6, eEGG7, and eEGG7-AI, flick the device 3 times to display the menu, then flick it once to switch to the next menu item, until you switch to the WiFi menu, then flick it twice.
3

Connect your phone to a 2.4G WiFi network.

4

Run the ESP-Touch app on your phone, and input your WiFi password, leave the AES Password field blank (do not enter anything).

5

Click "Confirm" button and wait for ESP-TOUCH to indicate successful configuration.

‚ö†Ô∏è Note 1: The device can only work in 2.4G WiFi mode. Please ensure your router supports 2.4G WiFi.

‚ö†Ô∏è Note 2: If network configuration fails on the first try, please try more several times.

Alternative Method: Edit config.ini File

1

Connect the eEGG device to your computer via USB cable.

2

Find the config.ini file in the ".set" directory of your eEGG device.

Note: The .set folder is hidden. Enable "Show hidden files" in your file explorer.

3

Open the config.ini file with a text editor and modify the wifi_SSID and wifi_password entries:

[SETTINGS]
default_feature = 0        ; Default power-on function
turnoff_confirm = 0        ; Power-off confirmation
shutdown_time = 60         ; Auto power-off time (minutes)
sensitivity = 4            ; Shake sensitivity (1-5)
bat_percent = 0            ; Show battery percentage
usbmsc_encrypt = 0         ; USB MSC encryption
wifi_ssid = yourWiFiSSID   ; Your WiFi SSID
wifi_password = 12345678   ; Your WiFi password
[END]

Transferring Files via Mobile Phone

1

Download and install the "eEGG File Conversion and Transfer Assistant" APP to your mobile phone.

Download APK

2

Enter mobile file transfer mode on the eEGG device.

3

Connect your phone to the same WiFi network (SSID) as your eEGG device.

4

Open the mobile app and select the eEGG model in the settings menu.

5

Scan the QR code on the eEGG with the mobile app to connect.

6

Select video, image, and music files from your phone and transfer them to the eEGG.

‚ö†Ô∏è Important Notes:
  • eEGG File Converter and Transfer Assistant will automatically convert image and video formats before transferring files
  • Only supports landscape format videos no longer than 60 seconds
  • Limit video resolution to 1920x1080 and frame rate to 30fps
  • Test network connection before transferring files

Transfer Files via USB

Connecting the device directly to the computer via USB allows for easy copying of media files to the device's memory.

‚ö†Ô∏è Important:
  • Before copying video and image files, you need to use conversion tools
  • Do not delete directories on the device
  • .set directory - essential UI and configuration files
  • .voice directory - voice response files (eEGG7 only)
  • game directory - game files (not in eEGG7-AI)
  • video/mjpeg directory - user video files
  • image directory - user image files
  • audio directory - user music files
