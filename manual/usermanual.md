## Operation Guide

### Power On 
There is a silicone power button on the top of the eEGG. Press this button firmly to power on the device.

‚ö†Ô∏è***Caution***: After receiving the eEGG, please fully charge the device before use.

### Shake-and-Do
Operation The eEGG has a built-in G-sensor and can be used to achieve different functions by **swinging** it. Gently shake the eEGG several times, then stop and wait for about 1 second. The device will recognize the number of shakes and execute the corresponding function:

|   Operation   | Function    |             Description                                                            |
|:-------------:|:-----------:|:-----------------------------------------------------------------------------------|
| Shake 1 time  | Next/+      | Switch to the next menu item; Play the next media file; Increase the volume.       |
| Shake 2 times | Run/Prev./- | Execute the selected menu or function; Play the previous file; Decrease the volume.|
| Shake 3 times | Open Menu   | Display the menu.                                                                  |
| Shake 4 times | Mode/Volume | Shaking 4 times opens a pop-up menu for adjusting volume/playback mode;            |
| Shake 5 times | Power Off   | After powering off, press the button on the eEGG to power it on.                   |

## Charging
#### 1Ô∏è‚É£ This device can be charged using various chargers with a Type-C interface. Under normal circumstances, the internal battery can be fully charged within 90 minutes.
#### 2Ô∏è‚É£ If the device has been left unused for an extended period, causing over-discharge of the internal battery, please plug in the charger for 10 minutes before trying to turn the device on.
#### 3Ô∏è‚É£ If the device is not used for a long time, it is recommended to fully charge it at least once every two months.

## <a id = 'wificonfig'>WiFi Config</a>
#### 1Ô∏è‚É£ Download **ESP-Touch** app. 
- [ESP-Touch for Android](https://github.com/EspressifApp/EsptouchForAndroid/releases/download/v2.0.0/esptouch-v2.0.0.apk)
- [ESP-Touch for iOS](https://apps.apple.com/cn/app/espressif-esptouch/id1071176700)
#### 2Ô∏è‚É£ Start "WiFi config" on eEGG:
- **For eEGG5**, flick the device 3 times to display the menu, then flick once to switch to the next menu item, until you switch to the WiFi menu, then flick twice. The device will show a WiFi configuration picture.
- **For eEGG6, eEGG7, and eEGG7-AI**, flick the device 3 times to display the menu, then flick it once to switch to the next menu item, until you switch to the WiFi menu, then flick it twice. The device will display the WiFi configuration icon.
#### 3Ô∏è‚É£ Connet your phone to a 2.4G WiFi network.
#### 4Ô∏è‚É£ Run the **ESP-Touch** app on your phone, and input your WiFi password, leave the **AES Password** field blank (do not enter anything).
#### 5Ô∏è‚É£ Click "Confirm" button and wait for **ESP-TOUCH** to indicate successful configuration.
- **‚ö†Ô∏èNote 1**: The device can only work in 2.4G WiFi mode. Please ensure your router supports 2.4G WiFi and that the SSID selected for one-click network configuration in WeChat is 2.4G.
- **‚ö†Ô∏èNote 2**: If network configuration fails on the first try, please try more serval times.

#### ‚ú≥Ô∏è If you are unable to download or use the app, you can also edit the config.ini file:
##### 1Ô∏è‚É£ Connect the eEGG device to your computer via USB cable. After successful connection, you will see a new "removable disk" on your computer.
##### 2Ô∏è‚É£ Find the config.ini file in the "**.set**" directory of your eEGG device. 
  **Note:** The .set folder is hidden.
  - **On Windows**: open File Explorer ‚Üí View ‚Üí Show ‚Üí Hidden items.
  - **On macOS**: in Finder, press Cmd + Shift + . (period) to reveal hidden files.
##### 3Ô∏è‚É£ Open the config.ini file with Notepad and modify the wifi_SSID and wifi_password entries, then save it.

~~~ 
[SETTINGS]
default_feature = 0		    ; Default power-on function: 0 (default): Function before power-off / 1: Play video / 2: Play picture / 3: Game  
turnoff_confirm = 0		    ; Whether confirmation is required before powering off: 0: No / 1 (default): Yes  
shutdown_time = 60		    ; Automatic power-off time (unit: minutes) (default 60 minutes, minimum 1 minute, maximum 240 minutes)
sensitivity = 4			      ; Jitter sensitivity (1~5, 1 is the highest sensitivity, 5 is the lowest)
bat_percent = 0			      ; Whether to display battery percentage 0 (default): No / 1: Yes
usbmsc_encrypt = 0		    ; Whether to enable USB MSC encryption 0 (default): No / 1: Yes
wifi_ssid = yourWiFiSSID	; Set your WiFi SSID, or you can set it up with one click via WeChat
wifi_password = 12345678	; Set your WiFi password
[END]
~~~

## <a id = 'tcpserver'>Transferring Files to the Device via Mobile Phone (for eEGG6, eEGG7, and eEGG7-AI)</a>
#### 1Ô∏è‚É£ Download and install the "**eEGG File Conversion and Transfer Assistant**" APP to your mobile phone (**Currently only supports Android phones**), [**APK Download Link**](https://zonestar3d.github.io/EggAPK/).
#### 2Ô∏è‚É£ Shake the eEGG 3 times to display the menu, then shake it once to switch to the next menu item until you switch to the WiFi menu, then shake it twice. The device will display the WiFi menu. Shake the eEGG again to select the item with the small mobile phone icon (![](tcp.png)), then shake it twice to enter the mobile file transfer mode. The device will display an icon with a QR code, below which the device's SSID and IP address are displayed.
#### 3Ô∏è‚É£ Connect your phone to the same WiFi network (SSID) as your eEGG device.
#### 4Ô∏è‚É£ Open the mobile app and select the eEGG model (EGG50/EGG60/EGG70) in the settings menu.
#### 5Ô∏è‚É£ Tap the camera icon in the mobile app to scan the QR code on the eEGG. The device's IP address will be automatically entered into the IP address field in the app. Then tap the network connection test icon; your phone should quickly display a successful connection message.
#### 6Ô∏è‚É£ Now you can tap the button on the mobile APK to select video, image, and music files from your phone and transfer them to the eEGG.
- **‚ö†Ô∏èNote 1**: eEGG File Converter and Transfer Assistant will automatically convert image and video formats before transferring files.
- **‚ö†Ô∏èNote 2**: eEGG File Converter and Transfer Assistant only supports landscape format videos no longer than 60 seconds.
- **‚ö†Ô∏èNote 3**: Some file formats may fail to convert. Please limit the video resolution to 1920x1080 and the frame rate to 30 frames per second when recording.
- **‚ö†Ô∏èNote 4**: Before transferring files, tap the network test icon on the app to test your network connection. If the connection fails, please check the following in order:
- Check the WiFi signal strength (RSSI displayed at the bottom of the device should be >= 80%). If the signal is weak, move the eEGG closer to the router.
- Restart your phone and the app.
- Check if the eEGG is being blocked by the router.

## <a id = 'usbmsc'>Transfer Files via PC (USB)</a>
Connecting the device directly to the computer via USB allows for easy copying of media files to the device's memory.

**‚ö†Ô∏èNote:** Before copying video and image files, you need to use [**Video & Image Converter Tools**](#tools) to convert them.    
**‚ö†Ô∏èNote:** Do not delete directories on the device, otherwise some functions may malfunction.   
- üìÅ The **.set** directory contains essential UI interfaces and configuration files. Neither the folder nor the files can be deleted!
- üìÅ The **.voice** directory contains essential voice response files. Neither the folder nor the files can be deleted! (eEGG7 only)
- üìÅ The **game** directory contains game-related files. Neither the folder nor the files inside can be deleted! (Not available in eEGG7-AI)
- üìÅ The **video or mjpeg** directory contains user video files. The folder cannot be deleted, but the files inside can.
- üìÅ The **image** directory stores user image files. The folder itself cannot be deleted, but the files inside can.
- üìÅ The **audio** directory stores user music files. The folder itself cannot be deleted, but the files inside can.
